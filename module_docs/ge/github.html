ge.github

GitHub data fetching via gh CLI.

Provides functions to retrieve issues, PRs, comments, timeline events, related commits, and referenced code – all through the gh CLI so that private repo access is handled by the user’s existing gh auth.

ge.github.find_related_commits(repo, issue_number, *, limit=20)[source]

Find recent commits that reference this issue number.

>>> # commits = find_related_commits('owner/repo', 42)
ge.github.find_related_prs(repo, issue_number)[source]

Find PRs that reference or close this issue.

Uses the timeline API to find cross-referenced PRs, and also searches for PRs mentioning the issue number.

>>> # prs = find_related_prs('owner/repo', 42)
